# ICMP-flooder
My own project which send ICMP packets to target IP-address (URL too) 

ICMP-flood.py - create and send ICMP-packets to target IP(URL too) address. Also you can specify port number, length of packets, and frequence.

  usage: ICMPFlood.py [-h] [-i] [-u] [-p] [-l] [-f]

  optional arguments:
    -h, --help  show this help message and exit
    -i          Enter target ip address of destination
    -u          Enter target url address
    -p          Specify port number
    -l          Specify packet length
    -f          Specify value of frequence to send packet

ICMP-Graphic.py - same script with graphic interface.
